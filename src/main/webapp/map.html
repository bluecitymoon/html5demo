gmaps.js

Google Maps API with less pain and more fun

Map events

GMaps.js allows to define map events very easily:

map = new GMaps({
  div: '#map',
  zoom: 16,
 lat:  31.230393,
  lng: 	121.473704,
  click: function(e) {
    alert('click');
  },
  dragend: function(e) {
    alert('dragend');
  }
});

NoteYou can check the list of map events here.
